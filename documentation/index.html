4. Installation

Quick Start

  1. Extract the downloaded ZIP file to your desired location
  2. Open the main-files folder
  3. Double-click index.html to preview in your browser
  4. Upload all files to your web server to deploy

Server Requirements

This is a static HTML template with no special server requirements. It will work on any web server that can serve HTML files:

  • Apache
  • Nginx
  • IIS
  • Any static hosting service (Netlify, Vercel, GitHub Pages, etc.)
Important: The contact form requires server-side processing. You'll need to integrate it with your preferred backend solution (PHP, Node.js, etc.) or a service like Formspree.

5. Customization

Colors

All colors are defined as CSS variables in style.css for easy customization:

:root { --primary-color: #1a4d6d; /* Main blue */ --primary-dark: #0f3550; /* Darker blue */ --accent-color: #c9a961; /* Gold accent */ --text-dark: #1f1f1f; /* Dark text */ --bg-light: #fafafa; /* Light background */ /* ... more variables */ }

Fonts

The template uses Google Fonts:

  • Montserrat - Body text and UI elements
  • Playfair Display - Headings and titles

To change fonts, update the Google Fonts import in each HTML file's <head> section.

Logo

The logo is currently text-based. To use an image logo:

<a href="index.html" class="logo"> <img src="path/to/your-logo.png" alt="Your Company Name"> </a>

Images

Replace Unsplash placeholder URLs with your own images:

  1. Upload your images to the assets/images/ folder (create if needed)
  2. Find and replace Unsplash URLs in HTML files
  3. Update the src attributes with your new paths
